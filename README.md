# Demo on Jboss Fuse

## Overview
This repository contains a collection of demos on Redhat Jboss FUSE ESB.

### Prerequisites for All demos

Before building and running examples you need:

* Maven 3.3.9 or higher
* JDK 1.8
* JBoss Fuse 6.3.0
* MySQL community edition server
